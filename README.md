# HR Equipment Django

Django проект для управления оборудованием сотрудников.

## Функциональность
- Учет сотрудников и оборудования
- Управление учетными данными (AD, Email, Битрикс24, SIP)
- Отслеживание статусов оборудования

## Установка
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
